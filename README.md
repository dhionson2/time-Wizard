# TimeWizard: Biblioteca para Manipulação de Datas e Horas em Java

## Descrição

TimeWizard é uma biblioteca Java que oferece funcionalidades para realizar operações com datas e horas de forma simples e eficiente. Com TimeWizard, é possível adicionar, subtrair, comparar, formatar e converter datas e horas de maneira conveniente.

## Funcionalidades

- **Adição e subtração de dias, meses, anos e horas:**
  - Adicione ou subtraia dias, meses, anos e horas a partir de uma data ou hora específica.
- **Comparação de datas e horas:**
  - Compare datas e horas para determinar qual é anterior, posterior ou igual.
- **Formatação de datas e horas:**
  - Formate datas e horas em diferentes formatos de string.
- **Conversão entre diferentes formatos de data e hora:**
  - Converta datas e horas entre diferentes formatos, como ISO 8601 e formatos customizados.
- **Cálculo de idade:**
  - Calcule a idade de uma pessoa a partir da data de nascimento.
